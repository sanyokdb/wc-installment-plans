# WC Installment Plans

Плагин для управления вариантами рассрочки в WooCommerce.

## Требования

- WordPress 6.9.1+
- WooCommerce 10.5.3+
- PHP 8.0+

## Установка

1. Скопируй папку `wc-installment-plans` в папку `/wp-content/plugins/`
2. Активируй плагин из админки WordPress
3. Перейди в WooCommerce > Планы рассрочки

## Использование

### Добавление плана рассрочки

1. В админке перейди в **WooCommerce** > **Планы рассрочки**
2. Нажми **Добавить новый**
3. Введи название плана (например, "Стандартный")
4. Установи логотип плана (необязательно)
5. Укажи комиссию для каждого срока:
   - 3 месяца
   - 6 месяцев
   - 9 месяцев
   - 12 месяцев
   - 18 месяцев
   - 24 месяца
6. Нажми **Опубликовать**

### Допустимые значения комиссии

- Минимум: 0%
- Максимум: 100%
- Допускаются значения с десятичной точкой (например, 2.5%)

## Структура данных

Каждый план рассрочки содержит:

- **Название** - Название плана
- **Логотип** - Изображение логотипа (используется в шапке WordPress)
- **Комиссии** - Процент комиссии для каждого срока
